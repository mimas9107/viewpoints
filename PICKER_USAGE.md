# 監視器選擇器使用說明

## 🎯 功能概述

監視器選擇器讓你可以從資料庫中輕鬆選擇監控點，並自動產生配置檔案。

## 📁 檔案說明

```
viewpoints/
├── index.html              # 主監控牆頁面
├── picker.html             # 監控點選擇器 ⭐ 新增
├── cameras_database.json   # 監控點資料庫 ⭐ 新增
├── viewpoints.json         # 當前配置
├── start-server.py         # 服務器啟動腳本
└── README.md              # 使用說明
```

## 🚀 使用流程

### 步驟 1：啟動服務器

```bash
cd viewpoints
python3 start-server.py
```

### 步驟 2：開啟選擇器

在瀏覽器開啟：
```
http://localhost:8000/picker.html
```

### 步驟 3：選擇監控點

1. **搜尋** - 在搜尋框輸入關鍵字（例如：「陽明山」、「國道一號」）
2. **篩選** - 點擊分類標籤篩選（國道、景點、市區等）
3. **選擇** - 點擊監控點項目即可加入選擇

### 步驟 4：匯出配置

1. 點擊「📥 匯出配置」按鈕
2. 下載 `viewpoints.json` 檔案
3. 將檔案覆蓋到專案根目錄

### 步驟 5：查看監控牆

重新整理 `http://localhost:8000`，你的自訂監控牆就會顯示！

## 🎨 界面功能

### 搜尋欄
- 即時搜尋監控點名稱或地點
- 支援模糊搜尋

### 分類標籤
- **全部** - 顯示所有監控點
- **國道** - 國道一號～十號
- **景點** - 國家公園、風景區
- **市區** - 六都市區道路
- **快速道路** - 台61線～台88線
- **省道** - 省道公路

### 已選擇面板
- 即時顯示已選擇的監控點
- 可單獨移除不要的項目
- 顯示選擇數量

### 操作按鈕
- **清空** - 清除所有已選擇的監控點
- **匯出配置** - 產生 viewpoints.json 檔案
- **預覽監控牆** - 顯示配置資訊

## 📊 資料庫擴充

### 新增監控點

編輯 `cameras_database.json`，加入新的監控點：

```json
{
  "id": "11",
  "name": "你的監控點名稱",
  "description": "詳細描述",
  "type": "image",  // 或 "youtube"、"hls"
  "imageUrl": "https://...",  // 根據類型選擇對應的 URL 欄位
  "location": "地點",
  "category": "分類",
  "tags": ["標籤1", "標籤2"]
}
```

### 三種監控類型

**1. 靜態圖片**
```json
{
  "type": "image",
  "imageUrl": "https://cctv1.freeway.gov.tw/abs2jpg.php?sn=..."
}
```

**2. YouTube 直播**
```json
{
  "type": "youtube",
  "youtubeId": "RttyIGHbN_w"
}
```

**3. HLS 串流**
```json
{
  "type": "hls",
  "hlsUrl": "https://.../.../live.m3u8"
}
```

## 💡 使用技巧

### 技巧 1：快速建立主題監控牆

**上班通勤：**
1. 搜尋「國道一號」
2. 選擇你經過的路段
3. 搜尋「市區」
4. 選擇公司附近路口

**假日出遊：**
1. 篩選「景點」
2. 選擇想去的國家公園
3. 篩選「國道」
4. 選擇往返路線

### 技巧 2：混合監控類型

建議搭配使用：
- 📷 靜態圖片 - 路況監控（定時刷新）
- 🎥 YouTube - 景點直播（實時串流）
- 📡 HLS - 市區路口（高品質實時）

### 技巧 3：最佳監控數量

- **2×2 (4個)** - 適合單一主題
- **3×2 (6個)** - 適合上班通勤
- **3×3 (9個)** - 適合多地監控
- **4×3 (12個)** - 適合完整路網

## 🔧 故障排除

### 問題：選擇器打不開

**解決：**
確認服務器是否啟動，並訪問 `http://localhost:8000/picker.html`

### 問題：匯出的配置無法使用

**解決：**
1. 確認檔名為 `viewpoints.json`
2. 檔案放在專案根目錄
3. 重新整理瀏覽器（Ctrl+Shift+R）

### 問題：找不到想要的監控點

**解決：**
1. 查看 [tw.live](https://tw.live) 找到監控點資訊
2. 手動加入到 `cameras_database.json`
3. 重新整理選擇器
